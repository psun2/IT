## React 와 Create ReactApp 의 차이점

Create ReactApp 은 개발자가 개발하기 편하도록 각종 라이브러리와 플러그인들이 미리 셋팅이 되어있는 프로젝트를 의미하며,  
보통 현업에서는 Create ReactApp 을 사용 하지 않고 React 프로젝트에 필요한 라이브러리를 설치하는 방식으로 업무를 한다고 합니다.
