# `Ubuntu Node.js LTS (defalut) 설치`

보기 힘들다면 [참고](https://blog.naver.com/tjddjs90/221879895190) 하시기 바랍니다.

## curl 설치하기

```
$ sudo apt-get install curl
```

## node.js LTS 설치하기

```
$ sudo apt-get install nodejs
```

​

## 버전 확인하기

```
$ node -v
$ npm -v
```
