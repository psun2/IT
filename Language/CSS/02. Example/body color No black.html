body 태그의 color: black == NONO!!

color: black

니콜라스 선생님께서 body 태그를 reset css로 초기화 하고 color 를 black 으로 하는건 굉장히 좋지 않은 습관 이라고 합니다. 그래서 최대한 black 은 지양하고 black 과 거의 유사한 color 를 지향 하라고 합니다.

위 글은 color: black 으로 작성된 글 입니다.

color: #202020

이 글은 color: #202020 로 작성 된 글입니다. black 은 아니지만, 사람의 눈으로 비교 할 수 없을 정도의 black 색상과 아주 유사한 색상입니다.
