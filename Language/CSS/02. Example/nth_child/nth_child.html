nth-child()

위 예제는 3개의 부모컨테이너 안에 다섯 개의 자식 컨테이너가 각각 있습니다.
위 예제들의 박스의 스타일은 class 이름이 box 인 것에만 스타일을 주었습니다.

예제를 진행하면서 느끼는 건데 스타일을 부여할때 선택자가 가장 중요한 부분 인 것 같습니다.

부모요소에 보라색의 background 는 box:nth-child(2) 로 작성 하였고,
자식요소에 연두색의 background 는 box :nth-child(2) 로 작성 하였습니다.
위 예제와 같이 선택자에서 스페이스 즉 띄어쓰기 하나에 따라 전혀 다른 결과 가 나옵니다.

box:nth-child(2) 의 의미는 class name이 box에서 두번째 요소를 선택한다는 의미이고
box :nth-child(2) 의 의미는 class name이 box의 하위 자식요소중 두번째 자식이라는 의미 입니다.

nth-child()의 경우 제가 지정한 box위에 다른 요소들이 존재하면, 그 요소 또한 counting이 되니 유의하시기 바랍니다. 실제로 box class 위에 다른 요소를 놓아 실험해 보세요.
