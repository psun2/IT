인라인 요소

span : 인라인 요소입니다. 수평적

span 태그는 기본적으로 인라인 요소입니다, 인라인 요소는 content가 수평적으로 쌓입니다. 인라인 요소는 크기와 높이를 설정 할 수 없습니다. 인라인 요소는 content의 크기 만큼의 box가 형성 됩니다. 또한, content가 없는 빈태그일 경우 크기와 높이 설정이 불가능하여, 보이지 않습니다.

블럭요소

div : 블럭 요소 입니다.
수직으로

div 는 기본적으로 블럭 요소입니다. 블럭 요소는 width 는 브라우저의 width와 동일합니다. 물론 크기를 지정해서 크기만큼 보이게 할 수 있으나, 그 라인(?) 을 전부 차지합니다. 그래서 content는 수직적으로 쌓 입니다. 크기와 높이를 지정할 수 없는 인라인 요소와는 반대로 블럭요소는 크기와 높이를 설정 할 수 있습니다. content가 없어도 크기와 높이를 가지기 때문에 브라우저 상에 표현 됩니다.

display: inline

display: inline

display: inline 를 담은 태그는 div 입니다. div는 블럭 요소이나 display-inline 을 통해 inline 요소로 변화 시킵니다. 인라인 요소와 마찬가지로 가진 내용 즉, content가 없으면, 화면상에 보이지 않습니다.

display: block

display: block display: block

이곳은 인라인 요소인 span 태그로 작성 했습니다. 인라인 요소인 sapn 태그를 블럭요소의 특징으로 변화 시켜주는 옵션(?) 속성 입니다.

display: inline-block

display: inline-block
display: inline-block

display: inline-block 은 인라인과 블럭요소를 합쳐놓은 속성입니다. 1. 크기를 가질 수 있습니다. 2. content 가 수평적으로 쌓입니다. 3. 크기를 가질 수 있으므로, 컨턴츠가 없어도 박스가 표현 됩니다. 4. content가 아무것도 존재 하지 않을시 전체적인 구조를 망가트립니다. (전체화면으로 보세요.)

visibility: hidden 과 none

1. display: inline-block
2. visibility: hidden
3. display: inline-block
4. display: inline-none
5. display: inline-block

display 속성은 아니지만, 알고 있으면 유용한 visibility: hidden 을 소개합니다. visibility: hidden 은 디스플레이를 숨기는 역할을 합니다. 하지만 자리는 그대로 차지하게 됩니다.

박스 모델의 4번인 보이십니까 ? 4번 박스는 코드상에 존재 합니다. display: none; 은 element 를 숨기는 역할을 하며, visibility: hidden 과는 다르게 자리 또한 차지 하지 않습니다.
