transition 효과를 사용 했습니다. ☜ over me!!!

transition 은 가상 클래스 선택자를 통해 사용 가능합니다. 예를 들면, 위 예제와 같이 :hover 이란 가상클래스 선택을 사용할때, 쓰여지며, 가상클래스를 통해 바꾸고자 하는 속성을 property로 줍니다. 가상 클래스 선택자가 아닌 클래스에서 transition 속성을 부여 합니다.

.box2 { transition: all 2s ease-in-out .5s } .box2:hover { font-size: 2rem; font-weight: bold; background-color: black; color: aqua; }

transition

속성 값 의미
transition-property 속성이름 전환 효과를 사용할 속성이름
all 모든속성에 적용
transition-duration 지속시간 몇 초간에 걸쳐 행동 할 것인지 정합니다.
s, ms 사용가능 1s 또는 1000ms
transition-timing-function ease-in-out 선택 사항입니다만, 효과가 미미합니다.
transition-delay 지연시간 몇 초후 동작이 시작될 것인지를 정합니다.
s, ms 사용가능 .5s 또는 500ms 0.5일시 앞에 0을 생략 가능합니다.
transition property duration timing-function delay 한 줄 작성이 가능합니다.

Example

.box { transition-property: background-color, color; transition-duration: 1s, .5s; transition-timing-function: ease-in-out; transition-delay: 3s; } .box { transition: width 1s, height 2s ease-in-out .5s; } .box:hover { color:red; background-color: blue; wdith: 300px; height: 300px; } 위의 .box의 두가지 방법중 편하신 방법으로 작성하시면 됩니다.
