box-shadow : 그림자

글 보다는 예제가 도움이 많이 되는 부분이므로 예제와 코드를 비교하며, 살펴봐 주세요.
.box { box-shadow: -3px -3px black; } 속성 값을 음수 즉, -3 을 주게되면, 좌측 과 상단에 그림자가 표현 됩니다. .box { box-shadow: 3px 3px yellowgreen; } 속성 값을 양수 즉, 3 을 주게되면, 우측 과 하단에 그림자가 표현 됩니다.
.box1 { box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.25); } 마지막 px 값은 그림자의 번짐효과를 추가 합니다.
.box2 { box-shadow: 0 0 4px red, 2px 2px 8px orange, 4px 2px 12px yellow; } 속성은 콤마로 구분하여, 두개 또는 그 이상의 속성을 연달아 사용할 수 있습니다.
