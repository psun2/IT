Transform(요소변화)

내 블로그에서 보기

이 예제만 눈으로 보실시 이해가 잘 안 가실 수 있습니다. transition 과 transform 이 같이 있는 예제 를 보시면 이해해 도움이 됩니다.

transform : 변환
translate : 옮기다.
rotate : 회전
scalie: 규모

transform: translate(값px, 값%) 로 작성합니다.
요소(element) 를 X 축 또는 Y축으로 옴겨줍니다.

transfrom: rotate()
요소(element) 를 z축(default), X축, Y축 으로 회전 시킵니다.
값은 0 ~ 360 의 수를 넣어 deg 단위를 쓰거나, 0.5turn 또는 1turn 으로 작성합니다.

transfrom: scale()
요소(element)의 크기를 1로 기준으로 줄이거나 키웁니다.
기본적으로 scale(x축, y축) 으로 작성하며, 단위는 없습니다. scaleX 의경우 가로 길이를, scaleY 의 경우 세로 길이를, scaleZ 는 perspective(단위는 px) (3D 공간을 만드는 원근감 포함) 과 같이 작성하여, 원근감 있는 규모를 수정합니다.
