# React 개발환경 구축

## 노드 설치 확인 및 버전 확인

```
node -v
```

## NPM 설치 확인 및 버전 확인

```
npm -v
```

## Yarn 설치

```
npm install yarn --global
```

## Yarn 설치 확인 및 버전 확인

```
yarn -v
```
