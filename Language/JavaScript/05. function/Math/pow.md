# Title : pow()

# Subtitle
숫자에 제곱하여 주자!!!!!!

# word
pow() : 거듭제곱 함수
base : 베이스, 기초, 기지
exponent: 멱지수

# 요약
Math.pow() 함수는 base를 exponent번 곱한 결과, 즉 base의 exponent승 을 반환합니다.

# 구문
Math.pow(base, exponent)

# 매개변수
[base]
밑, 즉 여러 번 곱할 수.

[exponent]
지수, 즉 곱하는 횟수.

[반환_값]
base의 exponent 제곱.

# 설명
pow()는 Math의 정적 메서드이므로, 사용자가 생성한 Math 객체의 메서드로 호출할 수 없고 항상 Math.pow()를 사용해야 합니다. (Math는 생성자가 아닙니다)
