<table></table> element 는 표를 만들 수 있는 element 로 <table>, <th>, <tr>, <td> 와 셋 트를 이루며 사용 됩니다. 먼저 element 하나 하나 살펴보도록 하겠습니다
  • <table></table>
    • 표를 만들겠다고 선언(?) 하는 기능 쯤으로 생각됩니다.
    HTML 에서 표를 작성할때 꼭 <table></table>로 감싸주어야 합니다.
    한마디로 표를 작성할때, 표에 들어있는 내용의 최상위 부모 element 라고 표현할수 있습니다.
  • <th></th>
    • table header 의 약자로 표의 제목(?) 기준(?) 쯤으로 됩니다.
    말로 설명하기엔 생각에 비해 글쓰기 능력이 부족함으로 예제로 대체 합니다.
  • <tr></tr>
    • table row 의 약자로서, 열을 담당합니다.
  • <td></td>
    • td 는 무슨 약자인지를 잘 모르나, column 즉, 행을 담당 합니다.
th 1 입니다. th 2 입니다. th 3 입니다.
tr1의 td1(1열 첫번째 data) tr1의 td2(1열 두번째 data) tr1의 td3(1열 세번째 data)
tr2의 td1(2열 첫번째 data) tr2의 td2(2열 첫번째 data) tr2의 td3(2열 첫번째 data)
tr3의 td1(3열 첫번째 data) tr3의 td2(3열 첫번째 data) tr3의 td3(3열 첫번째 data)

위의 소스코드를 살펴보겠습니다. 
        
            <table>
                <th>th 1 입니다.</th>
                <th>th 2 입니다.</th>
                <th>th 3 입니다.</th>
                <tr>
                    <td>tr1의 td1(1열 첫번째 data)</td>
                    <td>tr1의 td2(1열 두번째 data)</td>
                    <td>tr1의 td3(1열 세번째 data)</td>
                </tr>
                <tr>
                    <td>tr2의 td1(2열 첫번째 data)</td>
                    <td>tr2의 td2(2열 첫번째 data)</td>
                    <td>tr2의 td3(2열 첫번째 data)</td>
                </tr>
                <tr>
                    <td>tr3의 td1(3열 첫번째 data)</td>
                    <td>tr3의 td2(3열 첫번째 data)</td>
                    <td>tr3의 td3(3열 첫번째 data)</td>
                </tr>
            </table>
