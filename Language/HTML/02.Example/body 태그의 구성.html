body 태그는 html 문서의 본문에 해당되는 부분이고, 사용자에게 보여지는 태그 입니다.
머릿글(?)에 해당됩니다.
본문에 해당하는 여러 요소들을 담을 수 있습니다. 문서내 main요소가 나오는 것은 1번만 허용합니다.
목적지로 이동할 수 있도록 링크를 별도로 모아둔 영역이다.
내용적 흐름과 구조를 만들기 위해 내용을 나누는 용도입니다. 비슷 한 내용을 담슴니다.
독립적으로 구성된 글을 표시합니다.
흔히 사이드바라고 부르는 영역으로 배너, 용어 설명, 관련 상품 등 본문 내용과 직접적인 관련성이 적거나 없는 내용으로 구성됩니다.
꼬리(?) 부분에 해당합니다.
아무런 의미를 가지지 않는 태그입니다.
div와 마찬가지로 아무런 의미를 가지지 않는 태그입니다. div와 span 은 주로 box 모델에 많이 사용 됩니다.
