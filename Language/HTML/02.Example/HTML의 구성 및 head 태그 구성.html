브라우저에게 html 문서임을 알려줍니다. html에 문서임을 시작하는 태그입니다. 속성 값으로는 lang 즉 language의 약자 언어를 선택하는 속성이있습니다. head 태그는 사용자에게 불필요한 정보 이므로, 보여지지 않으며, 브라우저에게 필요한 정보를 제공합니다. html 문서의 정보를 알려줍니다. meta 태그의 charset 속성은 이 html 문서의 인코딩 방식입니다. 인코딩이라는 뜻은 잘 모르지만 즉, 우리가 작성한 코드를 컴퓨터가 알아들을수 있는 UTF-8 로 바꾸어줍니다. viewport 는 웹페이지의 크기를 결정한다고 합니다. author 는 작성자를 표시합니다. 검색시 요약된 정보(?) 를 표출합니다. Document 이 html문서의 제목 또는 주제 입니다.
