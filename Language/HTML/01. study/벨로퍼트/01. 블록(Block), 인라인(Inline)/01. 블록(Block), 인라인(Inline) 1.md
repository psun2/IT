# HTML - 블록 레벨(Block level) 요소와 인라인(Inline) 요소

1. 블록 요소
    1. DIV, H1, P
    2. 사용 가능한 최대 가로 너비를 사용한다.
    3. 크기를 지정할 수 있다.
    4. (with: 100%; height:0; 으로 시작한다.)
        높이가 0으로 시작 되고, 태그 안 의 글자 크기 만큼의 높이를 가진다.
2. 인라인 요소
    1. SPAN, IMG
    2. 필요한 만큼의 너비를 사용한다.
    3. 크기를 지정할 수 없다.
    4. (width:0; height:0; 으로 시작한다.)
