아주 식상하리 만큼 진부한 레이아웃 입니다.

많은 개발자들은 스크롤을 밑으로 하면, 네비게이션을 항상 볼 수 있도록 네비게이션이 위에 붙어있길 원할 것 입니다.

원했던 바로 그 기능이 "sticky" 입니다.

이 기능을 위해 자바스크립트를 이용해서 ... normal -> fixed 변경하고

Display block, position relative 설정을 통해 해당 기능을 구현했습니다.

그러나 "sticky" 라는 녀석을 css 속성 값으로 주어, 쉽게 구현할 수 있습니다.
