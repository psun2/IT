DOM 과 HTML

DOM은 HTML 의 상위개념(?) 입니다.

DOM

문서 객체 모델(The Document Object Model, 이하 DOM)

즉, input은 type을 가질수 있으며, type으로는 text, password 등으로 이루어진 Node 객체 입니다.

HTML 안에는 여러가지 노드 객체가 존재 하며, HTML도 여러가지 Node 객체를 다룰 수 있는 하나의 객체의 개념이며, 객체 형태로 작성된 문서 즉, 이를 통칭하여 일명 DOM (문서 객체 모델(The Document Object Model) 이라 합니다.)
