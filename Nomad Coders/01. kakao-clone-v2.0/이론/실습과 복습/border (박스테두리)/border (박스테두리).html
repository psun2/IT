border 속성 주기

border: 1px solid balck => 상, 하, 좌, 우 에 1px의 굵기의 검정 실선을 그어줍니다. border-bottom: 1px solid balck => 아래 에 1px의 굵기의 검정 실선을 그어줍니다. 마찬가지로 top, right, left 각각 줄수 있습니다. dashed 는 실선이 아닌 점선으로 표기하는 속성입니다.

border-radius 로 원그리기

border-radius 는 border 즉 경계를 뭉뚱그립니다. border-radius 의 값이 크면 클 수록 원에 가까워 집니다.
