color - inherit 과 initial

color: inherit;

inherit
Represents the computed value of the property on the element’s parent.
요소의 부모에서 계산 된 속성 값을 나타냅니다.

확실한 효과를 주기위해 a태그로 설명을 드리겠습니다.
태그는 a태그 즉 링크이므로, blue의 색상 계열이나, 링크가 클릭 된 적이 있으면 violet(?) 색상 계열이 나타나게 됩니다.
하지만 color: inherit; 속성으로 부모 body의 색상으로 변환 됩니다. a 태그가 부모의 color 값으로 변경됩니다.

color: inherit;

initial
Represents the value specified as the property’s initial value.
속성의 초기 값으로 지정된 값을 나타냅니다.

확실한 효과를 주기위해 a태그로 설명을 드리겠습니다.
a태그의 초기 값인 blue 계열의 색상이 나올꺼라 생각했는데, black (#000000) 색상이 나옵니다.
