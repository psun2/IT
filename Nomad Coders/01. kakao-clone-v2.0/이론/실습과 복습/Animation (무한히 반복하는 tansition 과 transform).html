Animation

transition 과 transform 을 이용해 가상 클래스 선택자에게 특별한, 일회성 애니메이션 효과를 주어 봤습니다.
하지만 효과는 일 회성이고 계속 지속 할 수 없었습니다.
하지만 밑의 예제처럼 애니메이션을 이용해 transform 및 애니 메이션 효과를 지속 시킬 수 있습니다.
구문
@keyframes animationName { form{ } to { } } from 과 to 를 넣는 방법 외에도. @keyframes animationName { 0%{ } 50% { } 100% { } } 또는 @keyframes animationName { 0%, 100%{ } 50% { } } 이렇게 합쳐서 여러 방법으로 시도 할 수 있습니다.
자세한 사항은 예제의 소스 코드를 참고 해주세요.
이렇게 애니메이션에 줄 효과를 마치 가상 클래스 선택자에 transform 등, 여러가지 효과를 입력 해주면 되고, 입력후 이 애니메이션을 사용하고자 하는 클래스의 스타일에서, animation 의 속성을 추가 해주고, 속성 값 또한 transition 과 매우 유사하다고 볼 수 있습니다. .box5 { box-shadow: 1px 1px 5px black; animation: 1s myAnimation3 infinite ease-in-out; } 위와 같이 애니메이션 속성을 추가해주고 그 값으로는, druation, 애니메이션 이름, infinite(무한 반복여부) ,timing-function 을 속성 값으로 주시면 되겠습니다.
잘 이해가 안가는 부분은 transition 과 transform, 응용방법 을 참고 해주시면 감사 하겠습니다.
