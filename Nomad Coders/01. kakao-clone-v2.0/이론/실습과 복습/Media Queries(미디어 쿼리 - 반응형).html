Media Queries(미디어 쿼리 - 반응형)

반응형이란 ?
우리가 만든 예쁜 웹 페이지는 모바일에서도 적상적으로 작동해야 됩니다. 하지만, 모바일로 접속해보시면, 가끔씩 위치도 않 맞고 기능도 정상 작동이 안되는 경우가 있습니다. 우리는 그런 일을 대비해 웹페이지를 반응형으로 만들어야 합니다. 이론의 설명만으로 저도 배운게 많이 없어 어려우니, Media Queries 즉, 반응형의 구문을 보도록 하겠습니다. @media screen and (min-width:320px) and (max-width: 640px) { * { background-color: tomato; color: black; } } @keyfarmes 애니메이션과 비슷하게 골뱅으로 선언 해줍니다.
@media screen 으로 선언 하며, and 뒤에는 조건 입니다. 즉 가로의 최소 사이즈가 320px, 그리고 가로의 최대가이즈가 640px 일때 배경색은 토마토 색 이고 글씨는 검정으로 변한다는 희미입니다. 선언된 media qeuries 안에서 우리는 style 태그에 css를 작성 하는 것처럼, 여러방법으로 스타일링 할 수 있습니다.

그리고 (min-width:320px) and (max-width: 640px) 에서 width 를 height 로 바꾸는등 조건을 수정할수 잇으며, (min-width:320px) and (max-width: 640px) and (min-height) and (max-height) 와 같이 조건을 추가 해 줄 수 도 있습니다.
