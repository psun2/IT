CSS 작성꿀팁

원하는 스타일이 안먹히는 경우!

  1. p태그 안에 div 를 꾸미려고 하는지 살펴 봅니다. (이유는 모르겠으나 p태그 내부의 div는 스타일이 안먹습니다.)
  2. >(직계자식 선택자) 와 space(하위 자손 선택자)를 혼동하여 작성했는지 살펴 봅니다.
  3. transition: all 1s esae-in-out .5s; 와 같이 속성을 한줄로 부여 할때는 값 사이 사이에 ,(콤마) 를 섞지 않습니다.
  4. scaile(.5, .5); translate(20px, 50%); 괄호 안에서 값을 두개 이상 작성할때는 ,(콤마 를 사용하여 값을 분리 시켜 사용 합니다.)
  5. trasnlate(20deg, 50); 과 같이 단위를 잘못 작성했거나, 생략한 경우가 있는지 확인 해 봅니다.
  6. background-color: red; 와 같이 ;(세미콜론) 으로 잘 닫아 주었나 확인 해 봅니다.
