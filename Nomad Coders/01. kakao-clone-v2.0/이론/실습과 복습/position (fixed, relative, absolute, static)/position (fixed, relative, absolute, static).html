참조

position: fixed

주의 : position: fixed를 사용시, 블럭 요소인 div도 width 크기를 줘야 나타나게 됩니다.

position: fixed 를 사용하면, 위치가 고정되어 아무리 스크롤을 해도, 변화하지 않습니다. top, bottom, left, right 값을 설정 할 수 있습니다.

position: relative

position: absolute

bottom: 5px;

right: 5px;

position: static

position: static => defalut 값 입니다.

position: absolute

position: relative 와 한 셋트로 움직이며, absolute 는 fixed 와 비슷하며, 부모 요소의 위치에 대한 관계를 확립합니다. position: absolute; => 부모요소 안에서 절대적인 위치를 취합니다. bottom: 5px; => 밑에서 5px right: 5px; => 오른쪽에서 5px 의 거리에 fixed 시킨다는 의미 정도가 됩니다.

position: relative

position: absolute 와 한 셋트로 움직이며, 자식요소가 absolute로, 위치의 움직임을 주려 할때, 부모요소에 position: relative 를 적용해, 자식요소가 움직일수 있는 기준을 마련해 줍니다.
